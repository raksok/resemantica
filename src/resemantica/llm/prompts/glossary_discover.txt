## INSTRUCTIONS
Identify all significant glossary-worthy terms from the source text above. These include:

- Character names and aliases
- Faction, sect, school, or clan names
- Location names (mountains, cities, regions, buildings)
- Martial techniques, skills, spells, or cultivation methods
- Items, artifacts, weapons, or treasures
- Realm or cultivation level concepts
- Creature or race names

For each term, provide:
- source_term: the exact Chinese text as it appears
- category: one of [character, alias, title_honorific, faction, location, technique, item_artifact, realm_concept, creature_race, generic_role, event]
- evidence_snippet: the surrounding context (roughly 40 characters around the term)

Exclude common vocabulary, generic verbs, common adjectives, and ordinary nouns. Focus on proper nouns and setting-specific terminology.
For example, these are NOT glossary-worthy:
- 二月二 — calendar date, not a proper noun
- 乡塾 — common noun ("village school"), not setting-specific
- 这时 — temporal conjunction, not a glossary term

## OUTPUT FORMAT
Return JSON only. Either a list or an object with key "glossary_terms".
If no terms match, return an empty list [] or {{"glossary_terms": []}}.
Do NOT wrap JSON in markdown code fences. Return raw JSON only.
Each row must include:
- source_term
- category
- evidence_snippet
