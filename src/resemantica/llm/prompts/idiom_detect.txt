## INSTRUCTIONS
Identify all Chinese idioms (成语、惯用语、俗语、谚语) from the source text above.
Focus on:
- Four-character idioms (成语)
- Common figurative expressions (惯用语)
- Proverbs and sayings (俗语、谚语)

For each idiom, provide:
- source_text: the idiom phrase only (e.g. "孤苦伶仃", NOT the full sentence "有位孤苦伶仃的清瘦少年")
- meaning_zh: the Chinese meaning/explanation of the idiom
- usage_notes: optional notes on how the idiom is used in context

Exclude:
- Calendar dates, time expressions, common nouns, regular descriptive phrases
- Terms that are not established idioms or set phrases

## OUTPUT FORMAT
Return JSON only.
If no idioms match, return an empty list [] or {{"idioms": []}}.
Do NOT wrap JSON in markdown code fences. Return raw JSON only.
Either a list or an object with key "idioms".
Each row must include:
- source_text (idiom phrase only)
- meaning_zh
- usage_notes (optional)
